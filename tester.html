USDC Code Tester

This HTML file should help you test your JavaScript.

Feel free to test your code as you wish. We will be evaluating your submissions in a testing script in our browsers that is similar to this.

We imagine you would use this file in the following way:

  1. Open this file in a browser.
  2. Open the web developer console.
    • If you have not done any work on the code yet, you should see two failing tests.
  3. Work your solution using a text or code editor of your choice.
  4. To iteratively test your work: 1) save your code, 2) flip back to this window, and 3) reload the page.
